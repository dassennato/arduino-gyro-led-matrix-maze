# Arduino - Maze Game w/ Led-Matrix and Gyro

A simple Arduino maze game using an 8x8 LED Matrix (MAX72XX) and an accelerometer/gyroscope sensor (GY-521/MPU-6050)

![](demo.gif)

## Components used

* 1 - Arduino UNO R3
* 1 - Max7219 module
* 1 - GY-521
* 1 - Breadboard
* 11 - M-M Wires
